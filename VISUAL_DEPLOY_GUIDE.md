## 🎯 ПОСЛЕДОВАТЕЛЬНОСТЬ НАЖАТИЙ:

1. **Railway.app** → `Login` → `Login with GitHub`
2. **Dashboard** → `+ New Project` → `Deploy from GitHub repo`
3. **Repos** → `telegram-bot-project-1` → `Deploy Now`
4. **Left Menu** → `Variables` → `+ New Variable` (6 раз для всех переменных)
5. **Left Menu** → `Deployments` → `View Logs` (проверить)
6. **Left Menu** → `Settings` → `Generate Domain`
7. **Browser** → `https://your-app.railway.app/health` (проверить)
8. **Telegram** → найти бота → `/start` (тест)

**Готово! Ваш бот работает 24/7! 🚀**
